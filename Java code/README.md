# README

## Java Application Project

### Overview
This is a basic Java application project.
The goal of this project is to practice object-oriented programming
and understand the structure of a Java application.

### Features
Basic class and object design

Simple business logic implementation

Clear package structure

### Technologies Used
Java

Standard Java libraries

### How to Run
1.Compile the project:

```bash
javac Main.java
```

2.Run the application:

```bash
java Main
```

### Notes

This project is created for learning purposes.
The focus is on understanding Java fundamentals rather than advanced frameworks.
